# Project 1 - Linux & Command Line Basics

## Objective
Demonstrate Linux filesystem navigation, file management, and understanding of the Filesystem Hierarchy Standard (FHS).

## Tasks Completed
- Filesystem navigation using pwd, cd, ls
- Created nested directories using mkdir -p
- Managed files using touch, cp, mv, rm
- Studied FHS directories:
  - /
  - /bin
  - /sbin
  - /etc
  - /var
  - /tmp
  - /home
  - /root
